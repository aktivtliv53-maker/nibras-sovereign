# 📘 العربية من الداخل: الهندسة الوجودية للتمكين البنيوي
### نظام التشغيل القرآني (Q-OS) | Qur’anic Operating System

---

## ⚡ نظرة عامة
**ليس تفسيرًا جديدًا، بل نظام هندسي مغلق يكشف قوانينه بنفسه.**
هذا المشروع هو أول نظام يحوّل الحرف العربي إلى وحدة طاقة قابلة للقياس، ويحوّل الجذر إلى معادلة رياضية يمكن اختبارها ونمذجتها عبر المحرك الإحصائي **Q-OS**.

---

## 🔬 المنهج العلمي والتقني
يعتمد النظام على تحليل إحصائي شامل لـ:
* **1713** جذرًا قرآنيًا.
* **6236** آية.
* **الطبقات السبع:** توزيع الحروف على المواقع الوظيفية (L1 / L2 / L3).
* **الروابط التساهمية:** قياس معاملات الانجذاب الفيزيائي بين الحروف داخل الجذور.

---

## 📂 محتويات المستودع
* `app.py`: المحرك البرمجي لتشغيل رادار البصيرة (Streamlit).
* `SCIENTIFIC_PAPER.pdf`: الورقة العلمية الرسمية الموثقة للمنهجية.
* `Data/`: جداول البيانات الإحصائية للروابط والمعاملات.

---

## 🎯 الرسالة
تحويل اللغة من "ظاهرة وصفيّة" إلى "هندسة وجودية" قابلة للقياس، حيث يلتقي النص بالرقم، والمعنى بالبنية.

---

### 🛡️ الملكية الفكرية
جميع الحقوق محفوظة للمؤلف **محمد صالح عبود**. هذا العمل محمي كـ "نظام تشغيل معرفي" وبروتوكول إحصائي غير مسبوق.
